# Self-balace
Self Balance Robot

A project for CPE338 | MICROPROCESSOR AND MICROCONTROLLER SYSTEM

A two-wheeled autonomous robot designed to maintain its upright position using a feedback control system.

    Hardware: Arduino UNO R3, MPU6050, DC Motor, IR
    Language: C++
    IDE: Arduino IDE
    Algorithm: PID Control (Proportional-Integral-Derivative)
